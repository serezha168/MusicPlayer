# Документация по музыкальному плееру

## Основные компоненты

### MainViewModel

Основной класс, отвечающий за логику приложения.

Ключевые свойства и методы:
- `IsPlaying`: Показывает, воспроизводится ли музыка в данный момент.
- `CurrentPosition`: Текущая позиция воспроизведения (от 0 до 1).
- `CurrentTime`: Текущее время воспроизведения в формате "мм:сс".
- `TotalTime`: Общая продолжительность трека в формате "мм:сс".
- `Volume`: Громкость воспроизведения (от 0 до 1).
- `IsUpdatingPosition`: Флаг, указывающий, обновляется ли позиция программно.
- `Play()`: Начинает воспроизведение.
- `Pause()`: Приостанавливает воспроизведение.
- `Stop()`: Останавливает воспроизведение.
- `OpenFile()`: Открывает диалог выбора аудиофайла.
- `SetPosition(double position)`: Устанавливает позицию воспроизведения.

### MainWindow

Главное окно приложения.

Ключевые элементы интерфейса:
- Кнопки управления: Play, Pause, Stop, Open File.
- Слайдер для перемотки трека.
- Слайдер для регулировки громкости.
- Отображение текущего и общего времени воспроизведения.

## Новый функционал

### Перемотка трека

Теперь пользователь может перематывать трек, используя слайдер. 

Реализация:
1. В `MainViewModel` добавлен метод `SetPosition(double position)`, который устанавливает новую позицию воспроизведения.
2. В `MainWindow.xaml` слайдер настроен на двустороннюю привязку к `CurrentPosition`.
3. В `MainWindow.xaml.cs` добавлен обработчик события `Slider_ValueChanged`, который вызывает `SetPosition` при изменении значения слайдера пользователем.
4. Добавлено свойство `IsUpdatingPosition` для предотвращения рекурсивных вызовов при программном обновлении позиции.

### Отображение времени

Добавлено отображение текущего времени воспроизведения и общей продолжительности трека.

### Регулировка громкости

Добавлен слайдер для регулировки громкости воспроизведения.

## Использование

1. Запустите приложение.
2. Нажмите "Open File" для выбора аудиофайла.
3. Используйте кнопки Play, Pause и Stop для управления воспроизведением.
4. Перемещайте слайдер для перемотки трека.
5. Регулируйте громкость с помощью соответствующего слайдера.

## Зависимости

Приложение использует библиотеку NAudio для работы с аудио.
